# Student Jobs Platform (University Project)

A university project website that helps connect students with job opportunities. The project focuses on building a clean and responsive user interface using Bootstrap, with basic Laravel backend integration.

## Features
- Responsive UI built with Bootstrap
- Job listing and job details pages
- Basic navigation and core user flows
- Simple Laravel integration (views/routing + basic form handling)

## Tech Stack
- **Backend:** Laravel (PHP)
- **Frontend:** Bootstrap (HTML/CSS/JS)
- **Local Environment:** XAMPP

- Contributed to **basic backend tasks** in Laravel (views, routing, and simple integration).
